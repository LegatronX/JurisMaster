## 🎓 Présentation

**JurisMaster** est une plateforme éducative juridique moderne conçue pour faciliter l'apprentissage du droit français. Développée par **Docteur Jus**, cette plateforme propose un catalogue de cours interactifs organisés par niveaux (Licence 1, Licence 2, Master) avec des fonctionnalités de gamification pour motiver les apprenants.

La plateforme combine rigueur académique et expérience utilisateur contemporaine pour rendre l'apprentissage du droit plus accessible et engageant.

## 🚀 Installation et utilisation

### Prérequis
- Un navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Une connexion internet (pour charger Tailwind CSS via CDN)

### Installation locale

1. Clonez le dépôt :
```bash
git clone https://github.com/LegatronX/JurisMaster.git
cd JurisMaster
```

2. Ouvrez `index.html` dans votre navigateur :
```bash
# Sur macOS
open index.html

# Sur Linux
xdg-open index.html

# Sur Windows
start index.html
```

### Utiliser via GitHub Pages

La plateforme peut être déployée via GitHub Pages en activant l'option dans les paramètres du dépôt.
